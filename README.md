# 🧮 Simple Calculator

A clean and responsive **Simple Calculator** built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with a modern UI design.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧠 Bracket support `( )`
* 🔢 Decimal calculations
* ❌ Clear button
* ⚡ Instant result calculation
* 🎨 Stylish UI with gradient buttons

## 🧑‍💻 Code Highlights

### Display Handling

* Replaces `0` with input
* Handles `Error` state properly

### Calculation Logic

* Uses JavaScript `eval()` for expression evaluation
* Handles invalid inputs with try-catch

---

## ⚠️ Note

`eval()` is used for simplicity.
In real-world applications, it's better to use safer parsing methods.
